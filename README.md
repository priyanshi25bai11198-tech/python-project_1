# Simple Library Management System

## Overview
This Python project is a simple Library Management System implemented with basic object-oriented programming. It allows users to add books, borrow books, return books, and view the current collection of books in the library. The system maintains an updated count of available books dynamically.

## Features
- Add new books to the library
- Borrow books if available
- Return borrowed books
- Display current list of books and total count

## Technologies and Tools Used
- Python 3.x
- Object-Oriented Programming (OOP) concepts

## Installation & Running the Project
1. Ensure Python 3 is installed on your system.
2. Download or clone the project files.
3. Run the main Python script in a terminal or IDE:
python your_script_name.py

text
4. Follow the prompts in the console for returning books or interacting with the library.

## Instructions for Testing
- Run the project.
- Observe the initial list of books displayed.
- When prompted, enter the name of a book to return it to the library.
- Verify the updated list and book count after returning a book.
- You can extend testing by modifying the script to borrow books or add more books.
